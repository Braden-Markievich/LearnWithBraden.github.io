Intro to Sequences

Sequences are another vital part of Calculus II. You will be given a function and then asked what value it will be ultimately heading to, along with determining if it's convergent or divergent. The growth rate rules discussed earlier in the growth rates page can be used when solving sequences. Examples are provided below.

Steps:
1. If applicable, try to apply growth rates to determine the value the sequence is approaching, since it is much faster this way
2. If you cannot determine it through growth rates, starting from 0, write out the first three terms of the sequence to try to see where the sequence is approaching
3. Determine if the sequence is convergent or divergent


Question 1: \[{-1^n}\]

Question 2: \[{3+cos(n)}\over n^2\]

Question 3: \[n^3 + 3n^2 \over 12n^3 + 4n^2 + 8\]
